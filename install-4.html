  1. 改進的安裝指南

    XLinux安裝手冊做了以下改進:

    • 改進了安裝章節

      加進了更多的細節和圖示, 讓使用者可以在簡單的閱讀完後, 就可以對Linux有初步的概略了解與操作.

    • 圖示說明

      我們採用圖示說明, 一個步驟一個步驟地解釋安裝和設定的過程.

    • 完全的套件清單

      第十二章包含了XLinux中每個套件清單明細. 每個細目包括套件的 中英文名稱, 類別 .套件清單是了解XLinux內容的最好方法.

    • 全新的常見問題

      第十四章我們將包含整理一組全新的常見問題. 如果您遇到難題(安裝前或後), 可參考此章節.

  2. 簡化與快速的安裝過程

    全程的安裝過程都有"上一步"和"下一步"的按鈕和整個安裝流程的選項, 可以解決許多'噢-- 我不想這麼做'的問題. 尤其最讓人感到好奇的，就是它的簡易性，及支援多國語言，我們將安裝的過程簡單化和安裝的流程程序化, 讓您在輕輕鬆鬆的在 25 分鐘之內完成安裝。

  3. 網路性能的提高

    • 視窗對話的Genie機智

      Genie 是個全方位的設定精靈, 它是網虎國際公司特別開發完成好用的系統設定工具。視窗對話方塊和簡單的操作方式, 讓您可以輕鬆快速地設定成您想要的XLinux系統, 對於系統管理者更是一項不可獲缺的好幫手喔!

    • 網路芳鄰 Samba File Server/Printer Server

      只要您在 Windows 98/95/NT 上設定的帳號名稱和 Linux Server 是一致的，那您 就可以直接透過密碼認證使用XLinux 當作您網路芳鄰的 File Server 與 Printer Server，另外，我們也預設了透過 [Samba/網路芳鄰] 的方式來管理您的 Web 網 頁與 CGI 等檔案，以及 Anonymous FTP Server 的檔案管理功能。您也可以查 看各式系統的運作記錄檔 (log)。

    • SMTP Mail Server/POP3 與 IMAP4 信箱管理環境

      基本上，當您在本機器上建了自己的帳號，您就同時擁有以這部機器為 Mail Sever 的電子郵件信箱，切記! 若您打算拿這部機器當作 Internet 上的 Mail Server，請記得要求您的 DNS 管理人員幫您將這部機器的 DNS 設定好。 本 Email Server 在 MS Windows 系列環境的設定方式完全不須改變。

    • Anonymous FTP Serve

      本 FTP Server 之管理亦可透過 Samba 網路芳鄰輕鬆地維護。

    • Kernel（系統核心)�

      Linux 系統核心統管所有硬體週邊之驅動程式、多 CPU 架構與記憶體管理，以及檔案系統....等，一 般化核心必須考量到多種可能性之模組化而佔用 不必要之系統資源，重新編譯往往是 Linux 專家 安裝完系統後為追求最佳化 Linux 之第一步驟， 透過特定專機規化方案，可在建制專機系統之初 便將 Kernel 調整到適切硬體與 CPU/Memory 使用 之最佳化狀態。

    • File Serving（檔案系統資源）

      Linux 系統隨機附加了許多類型的網路檔案系統， 諸多檔案系統最常應用到的便是 NFS（Network File System）與 SAMBA（SMB 網路芳鄰檔案系 統）。 穩定而快速的 NFS Server 與 Samba Server 可提升 辦公室應用場合之整體工作效率。

    • Networking（網路效能）

      Linux Kernel 2.2 剛開始採用 Net 4 架構，以更嚴 謹之網路規畫建構更有效率之網路系統，網虎 XLinux 針對 Routing、DNS 查詢、TCP 封包反應情況作細部設定，配合 Kernel 內提升 TCP / IP 效能，可達到更佳之網路效能。

    • IP-Chains Firewall

      如果您是透過本機器以 PPP 的方式撥接或固接連至 Internet，則您可以讓同一 網域的其他電腦直接透過它連上 Internet，不需再設定組態。

    • WWW Server

      找一個流覽器輸入本機器的網址，或若 DNS 管理人員尚未加進這部機器的設定 值時可以 IP 代替，您可以看到這部 Web Server 已經可以用了，您可以把網頁 存放在本 Server 透過網路芳鄰看到的 www_doc 目錄中，Web Server 立刻可以 看到更新過的網頁資料，同樣的，www_cgi 表示 CGI 程式存放的目錄，一樣可 以透過網路芳鄰來維護。�

    • PPP 自動撥接上網

      在XLinux的Genie設定精靈中您可以很容易地設定PPP 自動撥接上網, 您可在XLinux後段安裝時選擇撥接上網，並正確設好 ISP 與帳號密碼等, 或在安裝完成後於在Genie設定精靈中設定.�
