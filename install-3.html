3. XLinux 的源起與改革

  1. 源起:

    由於網虎公司工程師均有5-10年以上之電腦經驗，有感於好的作業系統 必須要好好 的推廣出去，因此我們研究，為何Linux這樣一個好的作業系統無法快速的普及 化? 分析結果發現，Linux 在安裝上面的進步目前已經到了可以接受的地步了，但 是在於『 設定 』上面卻還有很高的門檻，尤其對於一般人而言，設定 Linux 簡直要 他們的命，所以公司集合眾資深工程師的力量，包裝了許多 RPM file，重點是要將 『 設定 』這個步驟 簡化 讓使用者，在經過基本的安裝以後就可以不用花時間的 『 自動 』將系統都設定起來，並且有鑒於 Open Source 的精神所以我們決定將這樣 的內容都 Open 出來，共享整個軟體業 也期待這樣的動作能夠讓我們的軟體業更進 步。

    　　　　　　　　　　　　　　　　…………………這就是 XLinux 的源起

  2. 改革:

    XLinux 將不斷的改進與開發, 以提供使用者最完美且穩定的Linux系統與 簡易的使用介面, 我們也將秉持著 Open Source (開放原始碼)的精神, 將改革 和開發出來的新技術公開.

    XLinux的演進:

    1997: 研發PowerLinux 0.1版本
    1998: 開發完成PowerLinux 0.1
    1999 8月: PowerLinux 0.99e 問市
    1999 10月: PowerLinux 改版完成, 正式更名 XLinux
    1999 11月: XLinux 1.0 推出
