12. XLinux的套件清單

12.1 套件明細

分類套件:

此區為XLinux包含的所有套件明細, 我們將套件分類成以下幾種類型:??

  1. Administration Utilities (管理)
  2. Base Utilities (基礎)
  3. Communication Programs (通訊程式)
  4. Development (發展工具)
  5. Documentation (文件)
  6. Editors (編輯器)
  7. Electronics
  8. Games (遊戲)
  9. Graphics (圖型)
  10. Ham Radio
  11. Interpreters (釋譯器)
  12. Libraries (函式庫)
  13. Mail (郵件)
  14. Mathematics (數學)
  15. Miscellaneous (其它)
  16. Network (網路)
  17. Newsgroups (新聞群組)
  18. Software restricted in the US (美國限制軟體)
  19. Old Libraries (舊函式庫)
  20. Other OS's and file systems (其它作業系統和檔案系統)
  21. Shells (系統殼)
  22. Sound (音效)
  23. TeX
  24. Text Processing
  25. Utilities (工具)
  26. Web Software (Web 軟體)
  27. X Window (X 視窗)
